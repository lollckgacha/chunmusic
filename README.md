# 천타버스 노래 맞히기

유튜브 뮤직비디오 일부를 음성만 들려주고 곡 제목을 맞히는 실시간 방송 연동 퀴즈 게임입니다.

## 플레이 방법

이 저장소를 GitHub Pages로 배포한 뒤, 배포된 URL을 열면 바로 플레이할 수 있습니다.

- **싱글플레이**: 혼자 듣고 직접 정답을 입력
- **스트리밍 모드**: 스트리머를 선택하면 방송 채팅과 연동되어 시청자와 함께 진행 (OBS 오버레이는 URL에 `?overlay&streamer=<id>` 를 붙여서 사용)

## 문제 데이터 관리

곡 목록은 Google Sheets + Apps Script로 관리합니다 (이 저장소에는 포함되어 있지 않음 — Firebase 쓰기 권한이 있는 시크릿 값이 들어있어 별도로 관리). 곡 추가/수정 방법은 관리자에게 문의하세요.

## 로컬에서 실행

별도 빌드 과정 없이 `index.html`을 정적 서버로 띄우기만 하면 됩니다.
